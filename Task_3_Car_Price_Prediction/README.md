# Car Price Prediction with Machine Learning

## Project Overview
This project is Task 3 of the Oasis Infobyte Data Science Internship. The goal is to build a machine learning model to predict the selling price of cars based on various features such as present price, kilometers driven, fuel type, and the age of the car.

## Objective
To train a machine learning model (Random Forest Regressor) to accurately predict car prices.

## Dataset Information
The dataset contains the following features:
- `Car_Name`
- `Year` (used to calculate car age)
- `Selling_Price` (Target Variable)
- `Present_Price`
- `Driven_kms`
- `Fuel_Type`
- `Selling_type`
- `Transmission`
- `Owner`

## Technologies Used
- Python
- Pandas (Data Manipulation)
- Scikit-learn (Machine Learning Model)
- Matplotlib & Seaborn (Data Visualization)
- Jupyter Notebook

## Algorithm Used
- **Random Forest Regressor**: This algorithm was chosen because it performs well with a mix of numerical and categorical data and does not require extensive feature scaling.
