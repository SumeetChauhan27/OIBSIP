# Email Spam Detection with Machine Learning

## Project Overview
This project is Task 4 of the Oasis Infobyte Data Science Internship. The objective is to build a machine learning model to classify emails or SMS messages into two categories: **Spam** and **Ham** (Non-Spam).

## Objective
To train a machine learning model (Naive Bayes Classifier) to accurately predict whether a given text message is spam or not.

## Dataset Information
The dataset contains a collection of SMS messages with their corresponding labels:
- `v1` (Category): Contains labels "spam" or "ham".
- `v2` (Message): Contains the text of the email/SMS.

## Technologies Used
- Python
- Pandas (Data Manipulation)
- Scikit-learn (Machine Learning & Feature Extraction)
- Matplotlib & Seaborn (Data Visualization)
- Jupyter Notebook

## Algorithm Used
- **Multinomial Naive Bayes**: This algorithm was chosen because it performs exceptionally well for natural language processing (NLP) tasks and text classification like spam detection.
