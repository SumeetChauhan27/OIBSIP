# Sales Prediction Using Python

## Project Overview
This project is Task 5 of the Oasis Infobyte Data Science Internship. The goal is to predict the sales of a product based on advertising expenditures across different media channels (TV, Radio, and Newspaper).

## Objective
To build a machine learning model (Linear Regression) that accurately predicts future sales based on advertising budgets.

## Dataset Information
The dataset contains the advertising budget spent on different platforms and the corresponding sales:
- `TV`: Advertising budget spent on TV.
- `Radio`: Advertising budget spent on Radio.
- `Newspaper`: Advertising budget spent on Newspaper.
- `Sales`: The sales of the product (Target Variable).

## Technologies Used
- Python
- Pandas & NumPy (Data Manipulation)
- Scikit-learn (Machine Learning Model)
- Matplotlib & Seaborn (Data Visualization)
- Jupyter Notebook

## Algorithm Used
- **Linear Regression**: This algorithm is ideal for predicting a continuous numerical value (sales) based on linear relationships with the given features.
