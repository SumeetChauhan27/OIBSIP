# Iris Flower Classification

## Project Overview

This is Task 1 of my Oasis Infobyte Data Science Internship.

Main file: `SumeetChauhan_Task1.ipynb`

In this project, I created a machine learning model to classify iris flowers into three species:

- Iris-setosa
- Iris-versicolor
- Iris-virginica

## Objective

The objective of this project is to predict the species of an iris flower using its sepal and petal measurements.

## Dataset Information

The dataset contains the following columns:

- SepalLengthCm
- SepalWidthCm
- PetalLengthCm
- PetalWidthCm
- Species

The `Species` column is the target column.

## Technologies Used

- Python
- Pandas
- Scikit-learn
- Jupyter Notebook

## Project Workflow

1. Loaded the Iris dataset.
2. Removed the unnecessary `Id` column.
3. Selected input features and target column.
4. Split the data into training and testing data.
5. Trained a Random Forest Classifier.
6. Predicted flower species.
7. Checked accuracy, classification report, and confusion matrix.

## Algorithm Used

Random Forest Classifier

Random Forest is a machine learning algorithm that uses many decision trees and combines their results to make the final prediction.

## Results

The model successfully classified iris flowers using the given measurements.
